# claude-skills

## 项目概述
Claude Code skills 仓库，每个 skill 来自真实生产项目。

## Skill 开发
当用户提到「创建新 skill」「写一个 skill」「优化已有 skill」「改进 skill」
「skill 评审」「skill 评分」时，参考 [docs/skill-development-guide.md](docs/skill-development-guide.md)。

## 项目结构
- `skills/` — 各 skill 目录，每个包含 SKILL.md + 可选 references/
- `docs/` — 开发指南和文档
